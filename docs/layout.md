# Layout
`pixss` defines the most necessary classes to construct the simple layout with the common used classes easily:
- `.topbar`
- `.sidebar`
- `.main`
- `.container`
- `.footer`

![](img/layout.png)

## `.topbar`
define the Topbar with `fixed` position in page header
```html
  <div class='.topbar'>
    <div>logo</div>
    <div>navi</div>
  </div>
```
